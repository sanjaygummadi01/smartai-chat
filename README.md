# SmartAI Chat

SmartAI Chat is a modern AI chat interface built using React and TypeScript.  
The application provides a clean and responsive conversational UI designed with a component-based architecture and modern frontend development practices.

The project demonstrates how modern React applications can be structured to build scalable and interactive user interfaces.
